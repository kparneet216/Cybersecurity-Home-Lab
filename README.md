# Cybersecurity-Home-Lab
## Overview

This project documents the design and implementation of a virtual cybersecurity home lab built using VMware Workstation. The environment was created to safely practice networking, vulnerability assessment, penetration testing, and packet analysis against intentionally vulnerable systems.

The lab provides hands-on experience with commonly used cybersecurity tools and demonstrates practical skills relevant to entry-level cybersecurity and IT roles.

## Objectives

- Build a secure virtual cybersecurity environment
- Practice vulnerability assessment
- Perform network scanning
- Analyze network traffic
- Learn penetration testing techniques
- Gain practical experience with industry tools
